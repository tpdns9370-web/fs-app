# Personal Access Token 생성 가이드

## 빠른 단계

### 1단계: GitHub에서 토큰 생성

1. **이 링크를 클릭하세요**: https://github.com/settings/tokens/new

2. **설정 입력**:
   - **Note**: `fs-app-upload` (아무 이름이나 괜찮습니다)
   - **Expiration**: `90 days` (또는 원하는 기간)
   - **Scopes**: 아래 항목들을 체크하세요:
     - ✅ `repo` (전체 체크)

3. **"Generate token" 버튼 클릭**

4. **토큰 복사**: 
   - 초록색 박스에 나타나는 토큰을 복사하세요
   - ⚠️ **주의**: 이 토큰은 한 번만 보여집니다! 복사해두세요
   - 예: `ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 2단계: 업로드 실행

터미널에서 다음 명령어를 실행하세요:

```bash
cd "/Users/sewoonkim/Desktop/Vibe Coding/fs-project2"
./upload_token.sh
```

토큰을 입력하라고 나오면, 방금 복사한 토큰을 붙여넣으세요.
